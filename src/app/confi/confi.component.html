Politique de confidentialité

Cette politique de confidentialité explique comment l'application Jangalma (plateforme éducative), collecte, utilise, partage et protège les données personnelles et confidentielles des utilisateurs. En utilisant notre application, vous acceptez les termes de cette politique.

Entité responsable

Cette application est développée par l'entreprise Jangalma, mentionnée sur la fiche Google Play de l'application. Pour toute question relative à cette politique, vous pouvez nous contacter via :

  • Email : {{email}}

Types de données collectées

Notre application collecte les données suivantes :

Données personnelles :

Lors de l’inscription, l’utilisateur fournit les informations suivantes : Nom, prénom, email, ville, téléphone, code téléphone, la classe de l'élève et son mot de passe.

Utilisation des données collectées

Nous utilisons les données collectées pour :

  • Fournir et améliorer les fonctionnalités de l'application.
  • Diagnostiquer et corriger des problèmes techniques.
  • Assurer une expérience personnalisée pour chaque utilisateur.

Partage des données

Vos données personnelles ne seront jamais vendues à des tiers. Nous partageons certaines informations uniquement dans les cas suivants :

  • Avec Firebase, pour le stockage et la gestion des données dans un environnement sécurisé.

Sécurité des données

Nous mettons en œuvre des mesures de sécurité pour protéger vos données contre les accès non autorisés, modifications, divulgations ou destructions. Ces mesures incluent :

  • Le cryptage des données transmises et stockées.
  • L’utilisation de protocoles sécurisés pour l’échange d’informations.

Politique de conservation des données

Nous conservons les données personnelles aussi longtemps que nécessaire pour fournir nos services ou pour d’autres fins essentielles comme le respect des obligations légales. Les utilisateurs peuvent demander la suppression de leurs données en contactant {{email}}.

Droits des utilisateurs

Les utilisateurs ont les droits suivants concernant leurs données :

  • Accéder à leurs données personnelles collectées.
  • Demander leur modification ou leur suppression.
  • Accéder aux cours et aux livres.

Accès à la politique de confidentialité

Cette politique de confidentialité est accessible à tout moment via une URL publique et active : https://jangalma.net/confidentialité/.

Mises à jour de cette politique

Nous pouvons mettre à jour cette politique de confidentialité pour refléter des changements dans nos pratiques ou pour d’autres raisons opérationnelles, légales ou réglementaires. Toute modification sera publiée sur cette page et prendra effet immédiatement après sa publication.
